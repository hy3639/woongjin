게시판 Import

  • STEP01 Excel 다운로드

    샘플파일 다운로드/데이터 입력 오른쪽 버튼을 클릭하여 Excel 샘플파일을 다운로드 받으세요

    아래 설명을 참고하여 Import할 데이터 파일을 작성해 주십시오.

    • 1. 다운받은 샘플 엑셀 파일에 등록할 데이터를 입력합니다.
    • 2. 입력할 데이터는 아래와 같습니다. 컬럼이 2개 이상일 경우 순서에 주의해주세요. (* : 필수 입력 컬럼)
      • * 부모게시판ID(없을 경우 root 입력) | 게시판ID | * 게시판명 | * 게시판 영문명 |
      • * 게시판 타입(0:일반게시판, 1:링크게시판-Import 불가, 2:카테고리)
    • 3. 필수 입력 데이터가 없거나 입력 데이터 길이가 초과한 경우 정상적인 Import가 되지 않습니다.
    • 4. 데이터에 쉼표(,)가 들어 있으면 비 정상적으로 인식될 수 있습니다.
    • 5. 이미 등록되어 있는 데이터는 중복되어 등록되지 않습니다. 또한, 링크 타입 게시판 Import는 지원하지 않습니다.
    • 6. 데이터를 모두 입력하신 후 파일을 저장하고 STEP 2를 수행합니다.
  • STEP02

    데이터 파일 선택

    • 1. 찾아보기 버튼을 클릭하여 Import할 데이터 파일을 선택합니다.
    • 2. 확인 버튼을 클릭하면 STEP 3의 목록에 파일의 내용이 출력됩니다.
  • STEP03

    데이터 저장하기

    • 1. 등록 가능한 목록에만 체크박스가 표시됩니다.
    • 2. 체크박스로 등록할 데이터를 선택하고 저장 버튼을 클릭하면 데이터 Import가 완료됩니다.
