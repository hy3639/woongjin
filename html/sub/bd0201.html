유의사항

게시판 추가 방법

  • 게시판 트리의 폴더에서 마우스 우클릭 하면 여러 형태의 폴더 추가 메뉴가 나타납니다.
  • 게시판 생성 : 게시물을 등록할 게시판을 생성할 때 선택하는 메뉴 입니다.
  • 카테고리 생성 : 게시물 등록용이 아닌 여러 게시판을 하나로 묶는 Dummy 폴더를 생성할 때 선택하는 메뉴입니다.
  • 링크 생성 : 특정 URL로 링크되는 폴더를 생성할 때 선택하는 메뉴입니다.
  • 추가할 게시판 정보를 입력하고 저장하면 처음 선택했던 게시판 폴더의 하위에 생성됩니다.

게시판 조회/수정/삭제 방법

  • 게시판 트리의 폴더를 클릭하면 상세정보를 조회하고, 수정/삭제 할 수 있습니다.
  • 게시판 트리의 폴더에서 마우스 우클릭 하면 수정/삭제 메뉴가 나타납니다.

게시판 이동 방법

  • 이동을 원하는 게시판 폴더를 선택하여 Drag & Drop으로 이동할 수 있습니다.
